# Mobile UI Example

A ground-up custom UI redesign. A small floating button (inspired by Eclipse) sits on screen at all times, in every menu and every level. Tapping it opens a fully custom, draggable window with:

- A header you can drag to reposition the window anywhere on screen
- Tabs (General / Display / About) with animated switching and a sliding accent indicator
- Custom animated toggle switches that slide and cross-fade color instead of showing a checkmark
- Premium-styled buttons with built-in press feedback and a color flash on tap
- All custom-drawn art: soft shadows, gradient fills, rounded corners, no stock GD popup textures

This mod exists purely as a reference for building a completely custom UI system with Geode.
